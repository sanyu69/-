# 💡 게임 비주얼 시스템 디자인 브리프 (V1.0)

## 1. 브랜드 & 컬러 시스템 확정: `Neon_Vibe_Palette`

네온 리플레이 컨셉에 맞춰, 강렬한 대비와 깊이감을 주는 네온 스타일의 색상 팔레트를 정의합니다.

| 요소 | 역할 | HEX 코드 (Primary) | HEX 코드 (Accent/Glow) | 설명 |
| :--- | :--- | :--- | :--- | :--- |
| **배경 (Background)** | 깊이감 및 분위기 조성 | `#0A0A1A` (Deep Indigo/Black) | `#1E0D34` (Dark Purple Depth) | 어둡고 깊은 밤하늘 느낌의 배경. 모든 네온 효과가 강조됨. |
| **주요 액션 (Primary)** | 핵심 버튼, 활성화 요소 | `#00FFFF` (Electric Cyan) | `#00EEFF` (Bright Aqua Glow) | 게임 내 주요 상호작용 및 긍정적 피드백에 사용. |
| **보조 강조 (Secondary)** | 정보 표시, 보조 UI | `#FF00FF` (Hot Magenta) | `#FF33CC` (Pink Neon Pulse) | 경고, 점수, 보상 등 시각적 주의를 요하는 정보에 사용. |
| **텍스트/경계선** | 텍스트 및 프레임 | `#FFFFFF` (Pure White) | `#AAAAFF` (Soft Cyan Outline) | 가독성을 확보하며 네온 효과를 은은하게 더함. |

### 💡 타이포그래피 시스템

*   **헤드라인/제목:** `Monospace` 계열의 디지털 느낌을 주는 산세리프체 (`"Orbitron"` 또는 유사한 고정 폭 서체).
*   **본문/정보:** 높은 가독성을 위해 깔끔한 산세리프체 (`"Inter"` 또는 `"Poppins"`).

## 2. UI/UX 와이어프레임 구조 (Core Flow)

게임의 핵심 루프를 고려하여, 최소한의 요소로 최대의 몰입감을 제공하는 레이아웃을 제안합니다. 화면 비율은 **16:9 또는 4:3** 표준 웹 게임 비율을 가정합니다.

### A. 메인 게임 플레이 화면 (Gameplay HUD)

| 영역 | 위치 좌표 (상대적) | 디자인 요소 및 기능 | 색상 적용 |
| :--- | :--- | :--- | :--- |
| **게임 영역 (Game Canvas)** | `(0, 0)` ~ `(100%, 100%)` | 실제 게임 액션이 발생하는 메인 공간. | 배경: `#0A0A1A` |
| **상단 정보 바 (Top Bar)** | `Y: 5%` | 현재 점수, 남은 시간/에너지 게이지 표시. | 텍스트: `#FFFFFF`, 경계선: `#AAAAFF` |
| **하단 인풋/액션 바 (Action Bar)** | `Y: 90%` | 플레이어의 주요 입력(이동, 공격) 및 메뉴 접근 버튼. | 버튼 배경: 반투명 네온 효과 (`#1E0D34` 오버레이), 활성화 시 `#00FFFF` |
| **보조 UI (Overlay)** | 화면 중앙 또는 모서리 | 게임 오버, 일시정지(Pause), 설정 메뉴 아이콘. | 아이콘은 `Hot Magenta (#FF00FF)`로 강조. |

### B. 메인 메뉴 화면 (Main Menu)

*   **구조:** 중앙 정렬 기반의 대칭적 레이아웃.
*   **배경:** 깊은 보라색 계열 (`#1E0D34`).
*   **요소:**
    1.  **게임 시작 버튼:** 가장 크고 밝게 강조 (`Electric Cyan: #00FFFF`). (클릭 시 애니메이션 효과 추가)
    2.  **설정 메뉴:** 보조 액션 바에 배치.
    3.  **캐릭터/세이브 로드:** 네온 라인 아트를 사용하여 구분.

## 3. 핵심 아트 에셋 디자인 상세 (UI 요소)

개발에 즉시 적용 가능한 핵심 UI 컴포넌트의 시각적 정의입니다.

### 1. 플레이어 HP/에너지 바 (Health/Energy Bar)

*   **형태:** 가로 직사각형, 모서리가 약간 둥근 사각형.
*   **배경 (Empty State):** 투명하게 배경을 비추는 네온 효과 (`#0A0A1A`의 미세한 발광).
*   **채워진 부분 (Health/Energy):** 내부를 채우는 그라데이션 또는 솔리드 컬러.
    *   **HP Bar:** `Electric Cyan (#00FFFF)`에서 `#AAAAFF`로 부드럽게 전환되는 선형 그라데이션.
    *   **보조 바 (에너지):** `Hot Magenta (#FF33CC)`의 강렬한 단색 채움.

### 2. 버튼 스타일 (Button Style)

모든 상호작용 요소는 네온 빛이 발산되는 듯한 **글로우 효과**를 기본으로 합니다.

*   **기본 상태:** 배경은 매우 어두운 회색 (`#1E0D34`). 테두리만 희미하게 표시 (`#AAAAFF`).
*   **호버/활성화 상태:** 마우스를 올리거나 클릭 시, 버튼 내부에서 지정된 네온 색상(`Electric Cyan` 또는 `Hot Magenta`)이 **확산되어 발광하는 듯한** 효과를 줍니다. (CSS `box-shadow`와 `text-shadow`를 적극 활용하여 구현)

### 3. 배경 디테일 (Background Detail)

*   게임 세계의 깊이를 표현하기 위해, 화면 가장자리에 희미하고 흐릿한 네온 라인 패턴을 배치합니다.
    *   **패턴:** 화면 테두리 주변에 `Soft Cyan (#AAAAFF)`로 매우 얇은 사각형 그리드 또는 물결 무늬를 반복 적용하여 아날로그적인 레트로 느낌을 강화합니다.
