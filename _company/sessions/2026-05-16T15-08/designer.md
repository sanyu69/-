## 🎯 게임 초기 컨셉 및 비주얼 시스템 브리프 (V1.0)

### 1. 핵심 게임 컨셉: '네온 리플레이' (Neon Replay)

**컨셉 정의:** 복잡한 스토리나 깊은 전략보다는, **즉각적인 반응(Instant Feedback)**과 **시각적 만족감(Visual Satisfaction)**에 중점을 둔 하이퍼 캐주얼 액션 게임. 플레이어는 최소한의 조작으로 최대의 시각적 경험을 얻고, '다시 한 번 하고 싶다'는 욕구를 자극합니다.

**타깃 적합성:**
*   **캐주얼 게이머:** 복잡한 학습 없이 직관적인 컨트롤과 화려한 비주얼로 즉시 몰입 가능.
*   **광고 기반 모델 최적화:** 밝고 대비가 강한 색상은 광고 배너와 게임 내 요소의 시인성을 극대화하여 사용자 클릭 유도에 효과적입니다.

### 2. 아트 디렉션 (Art Direction)

**스타일 키워드:** **Neo-Retro Futurism (네오-레트로 미래주의)**
과거 레트로 게임의 아날로그적인 질감과 현대 네온 스타일의 극단적인 대비를 결합하여, 친숙함과 세련됨을 동시에 제공합니다.

| 요소 | 디테일 설명 | 디자인 의도 |
| :--- | :--- | :--- |
| **레이아웃** | 미니멀리즘 기반. 게임 플레이 영역(Canvas)은 중앙에 배치하고, 주변 환경(배경)은 단순화하여 핵심 액션에 시선이 집중되도록 설계합니다. | 웹 환경에서 로딩 속도와 모바일 반응성을 최우선으로 확보합니다. |
| **캐릭터/객체** | 픽셀 아트의 디테일과 네온 빛의 광원을 강조합니다. 형태는 단순하되, 그림자(Glow) 효과를 극대화하여 입체감과 에너지감을 부여합니다. | 캐주얼한 접근성을 유지하면서도 '네온'이라는 핵심 테마를 시각적으로 강력하게 전달합니다. |
| **UI/UX** | 최소한의 텍스트 사용. 점수, 생명력 등 필수 정보는 네온 색상으로 강조하여 게임 플레이 중에도 정보를 즉시 인지할 수 있게 합니다. | 복잡한 메뉴나 설명 없이 본질적인 '재미'에 집중하게 만듭니다. |

### 3. 컬러 팔레트 시스템 (Color Palette System)

**시스템 이름:** `Neon_Vibe_Palette`
색상 사용의 핵심은 **극단적인 보색 대비(High Contrast)**와 **채도 높은 네온 색상**의 조합입니다. 이는 웹사이트에서 시각적 주목도를 높이고, 모바일 환경에서도 명확하게 구별되도록 합니다.

| 역할 | 컬러 이름 (HEX Code) | RGB / HSL 값 | 사용 목적 및 심리학적 효과 |
| :--- | :--- | :--- | :--- |
| **Primary Accent** (핵심 액션) | `#00FFFF` (Cyan/Aqua) | RGB: 0, 255, 255 / HSL: 180, 100%, 100% | 에너지, 속도, 활력. 플레이어의 주된 상호작용 요소(조준선, 공격 효과)에 사용되어 즉각적인 반응을 유도합니다. |
| **Secondary Accent** (보조/경고) | `#FF00FF` (Magenta/Pink) | RGB: 255, 0, 255 / HSL: 300, 100%, 100% | 흥분, 위험, 보상. 코인 획득, 보스 등장, 성공적인 레벨업 등 긍정적 피드백에 사용합니다. |
| **Background Base** (배경) | `#0A0A1A` (Deep Indigo/Black) | RGB: 10, 10, 26 / HSL: 240, 5%, 10% | 깊이감과 대비 극대화. 네온 색상이 배경 위에서 폭발적으로 빛나도록 하여 시각적 임팩트를 높입니다. |
| **Neutral/Text** (정보) | `#FFFFFF` (Pure White) | RGB: 255, 255, 255 / HSL: 0, 0%, 100% | 핵심 정보(점수, 텍스트)를 명확하게 전달하기 위한 대비색으로 사용합니다. |

### 4. 타이포그래피 가이드 (Typography Guide)

**폰트 선택:**
*   **헤드라인/액션 텍스트:** **"Press Start 2P"** 또는 **"Orbitron"** 계열의 산세리프체 중 네온 느낌이 강하고 기하학적인 폰트를 사용하여 미래적이고 디지털적인 분위기를 강조합니다. (웹 환경에서 높은 가독성을 유지해야 함)
*   **본문/정보:** 시스템 메시지나 작은 정보는 **"Roboto Mono"** 또는 **"Space Mono"**와 같은 모노스페이스 계열을 사용하여 기술적이고 정밀한 느낌을 더합니다.
