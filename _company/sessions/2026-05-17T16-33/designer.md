# 💎 IAP 기반 비주얼 시스템 명세 (Basic $\to$ Pro 전환)

**목표:** $R_{Stability}$ 확보에 대한 신뢰를 바탕으로, Basic에서 Pro로의 전환이 '기능 해제'가 아닌 **'무결점 경험(Flawless Experience)'을 위한 프리미엄 접근권**임을 시각적으로 명확히 전달한다.

## 1. 브랜드 비주얼 시스템 재확인 (Foundation)

이전 세션에서 정의된 핵심 비주얼 규칙은 $R_{Stability}$를 반영하여 **'Neon Vibe'의 안정화 버전**으로 조정됩니다.

| 요소 | 상세 정의 | 적용 이유 |
| :--- | :--- | :--- |
| **컬러 팔레트 (Primary)** | Neon Blue: `#00FFFF` (안정성, 기술) / Deep Violet: `#6A0DAD` (프리미엄, 깊이) / Dark Gray: `#1A1A1A` (배경 안정성) | 네온의 활기로움(Vibe)을 유지하되, 어두운 배경과 차분한 보조 색상으로 **신뢰감**과 **프리미엄**을 확보한다. |
| **타이포그래피 (Font)** | Primary: 'Orbitron' (강렬함, 미래지향적) / Secondary: 'Roboto Mono' (코드/시스템 안정성 표현) | 게임의 네온적인 분위기를 유지하면서도, 시스템과 결제 로직의 **정확성과 안정성**을 강조한다. |
| **핵심 레이아웃 원칙** | **Grid & Symmetry:** 모든 UI 요소는 완벽한 그리드에 기반하며 대칭적 배치(Symmetry)를 최우선으로 한다. | $R_{Stability}$ 확보라는 목표에 맞춰, 시각적인 혼란을 최소화하고 **사용자 경험의 일관성**을 극대화한다. |

## 2. 수익화 흐름 (Monetization Flow) 시각적 요구사항

전환 흐름은 사용자가 자신의 안정성 레벨에 따라 자연스럽게 업그레이드하도록 설계되어야 한다.

### A. Basic 상태 (Stability Focus)
*   **시각적 특징:** 기능은 제한적이나, UI는 **최대한 깔끔하고 안정적**으로 표현한다. '잠금'이나 '제한'의 느낌을 최소화하고, 다음 단계에 대한 기대감을 조성한다.
*   **UI 요소:** 인앱 구매 버튼(CTA)은 존재하되, 시각적 강조(Glow effect)는 **낮게 설정**하여 불안정성을 간접적으로 암시한다.

### B. Pro 상태 (Premium Experience Focus)
*   **시각적 특징:** $R_{Stability}$가 보장됨을 시각적으로 표현한다. 색상 팔레트의 채도와 네온 효과를 **최대치로 증폭**시키며, 더 깊고 풍부한 그림자와 입체감을 부여하여 '프리미엄' 경험을 제공한다.
*   **UI 요소:** 전환 버튼은 명확하고 강력하게 강조되며, Pro 기능이 해제되었을 때의 시각적 보상(예: 새로운 아트 에셋 활성화)을 즉시 보여준다.

### C. 전환 UI (The Transition Gate)
*   **디자인 목표:** **신뢰와 가치 전달**. 단순한 결제가 아닌, '안정성 레벨 업'이라는 인식을 제공해야 한다.
*   **요구사항:**
    1.  **안정성 표시:** 전환 화면에 현재 시스템 안정성 상태(예: "Stability Verified")를 시각적 아이콘이나 텍스트로 명확히 표시한다.
    2.  **가치 비교:** Basic과 Pro의 차이를 **아트 에셋의 질감 변화**와 **네온 효과의 강도 차이**로 직관적으로 보여준다. (예: Pro에서는 더 깊은 네온 광선, 더 부드러운 그림자)

## 3. 아트 에셋 우선순위 정의 (Asset Prioritization)

아트 에셋 제작 시, $R_{Stability}$ 확보에 직접적인 영향을 미치는 요소부터 우선순위를 설정한다.

| 순위 | 아트 에셋/요소 | 세부 명세 및 요구사항 | 비고 ($R_{Stability}$ 연관성) |
| :--- | :--- | :--- | :--- |
| **P1 (Critical)** | **결제 성공/실패 애니메이션** | 결제 성공 시: 안정적인 녹색 네온 플래시. 실패 시: 경고 붉은색 및 시스템 메시지(Error Message)의 명확한 표시. | **최우선.** $R_{Stability}$에 대한 즉각적인 피드백 제공. |
| **P2 (Core)** | **티어별 UI 프레임** | Basic/Pro 상태 전환 시, 배경 색상 팔레트 및 전반적인 Glow 효과의 차이. | 사용자 경험(UX)의 핵심. 안정성 레벨에 따른 시각적 보상을 제공한다. |
| **P3 (Feature)** | **프리미엄 아트 에셋** | Pro 전용으로 잠금 해제되는 고해상도 배경, 아이콘 세트, 특수 이펙트(Particle Effects). | 수익화의 직접적인 동기 부여. $R_{Revenue}$ 달성에 기여한다. |
| **P4 (Aesthetic)** | **메인 메뉴/HUD** | 전체 디자인 가이드라인을 따르되, P1~P3가 구현된 후 시각적 완성도를 높이는 데 사용한다. | 미학적 완성도 향상 단계. |
